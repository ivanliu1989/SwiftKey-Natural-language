Part 2: Common phrases in text files

Our predictive text model will eventually depend on a collection of phrases and their frequencies from our text sample. For this Milestone Report, we will list the most frequent phrases composed of one to three words in samples drawn from the text (1-grams through 3-grams). We remove some very common stopwords (e.g. a, and, the) from the sample before the analysis.

Part 3: Predictive Text Model Algorithm

Here we discuss our plan for building an algorithm for predicting text. Since we have to consider the size and speed of our model, we will focus on 1-grams through 4-grams for the model for now. As we learn more about text-mining and language-modeling, our algorithm will probably change.

After checking that the user inputs a valid phrase (i.e. 2 words or more, text only) and cleaning up their input (i.e. alphabet characters only, lowercase, punctuation) we plan to use this algorithm:

  1. If the input phrase is 2 words, choose the most frequent phrase from among 3-word phrases that start with the input phrase. The predicted word is the last word of the chosen 3-word phrase.
  2. If the input phrase is 3 words, choose the most frequent phrase from among 4-word phrases that start with the input phrase. The predicted word is the last word of the chosen 4-word phrase.
  3. If the input phrase is longer than 3 words, use only the last 3 input words and follow the procedure for a 3-word phrase.
  4. If there is no prediction for a 3-word input phrase, use only the last 2 input words and follow the procedure for a 2-word phrase.
  5. If there is no prediction for a 2-word input phrase, select the most frequent word in the text sample.

Example: user input = we had a

most frequent phrase beginning with user input = we had a great

predicted word = great

Other considerations:

  • removing other common words and profanity

  • using word stems to reduce size of the Document Term Matrix
